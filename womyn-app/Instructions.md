WOMYN PROJECT - SETUP GUIDE
---------------------------
Prerequisite: Node.js (https://nodejs.org/)

to check if installed: ```node -v``` in CMD/Terminal

TO RUN ON WINDOWS:
Double-click "Run.bat".

TO RUN ON MAC:
1. Open Terminal.  
2. Type '```cd```' and drag this folder into the terminal.
3. Type: ```chmod +x Run.sh```
4. Type: ```./Run.sh```
